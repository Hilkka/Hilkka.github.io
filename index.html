TAMK Class Helper

January, 2016 for iOS

October, 2015

I participated in this hackathon in group of four and we won the second price with our idea of optimizing bus traffic with arduino boards. Read more!

Findkino

May, 2015 for Android

Findkino is a simple Android app I created for browsing the movie schedule of Finnkino movie theatres. It uses XML data provided by Finnkino and JSON from OMDb API to show the imdb reviews of the available movies.

Roommate

January, 2015 for Windows 8.1

Roommate is an app for Windows 8.1 operating system and can be used both in desktop and tablet. It shows all the available rooms or reservations in a spesific room in TAMK. The app uses json from TAMK Open data API.

Pailride

May, 2014 for Nokia Asha

Pail Ride is a mobile game for Nokia Asha. It was made as a school project in a group of 5 students and I was the graphical designer of our team. Unfortunately the game is no longer available for download.
